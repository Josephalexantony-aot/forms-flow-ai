# Template for commit messages
# Available variables:
# {ticket} - The ticket number (e.g., FWF-3229)
# {type} - The type of change (Feature, Bugfix, Hotfix, etc.)
# {description} - The main commit description
# {body} - Optional bullet points for multiple changes (optional)

# Single line format:
{ticket}: [{type}] {description}

# Multi-line format (when body is provided):
{ticket}: [{type}] {description}
{body}
